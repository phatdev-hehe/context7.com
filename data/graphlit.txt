TITLE: Setting Environment Variables for Graphlit Client
DESCRIPTION: Example of setting environment variables in a .env file for configuring the Graphlit Client. Includes environment ID, organization ID, and JWT secret.

LANGUAGE: env
CODE:
GRAPHLIT_ENVIRONMENT_ID=your_environment_id_value
GRAPHLIT_ORGANIZATION_ID=your_organization_id_value
GRAPHLIT_JWT_SECRET=your_jwt_secret_value

----------------------------------------

TITLE: Installing Graphlit Client with npm
DESCRIPTION: Command to install the Graphlit Client using npm package manager.

LANGUAGE: bash
CODE:
npm install graphlit-client

----------------------------------------

TITLE: Installing Graphlit Client with Yarn
DESCRIPTION: Command to install the Graphlit Client using Yarn package manager.

LANGUAGE: bash
CODE:
yarn add graphlit-client
