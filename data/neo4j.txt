TITLE: Running Neo4j Graphviz Visualization CLI
DESCRIPTION: Instructions for running the command-line interface tool to visualize Neo4j graph databases using Graphviz. The tool provides help information when run with the --help flag.

LANGUAGE: bash
CODE:
./graphviz --help
